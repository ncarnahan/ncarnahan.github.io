Rendering Engine

Simple rendering engine written in C++ using OpenGL 4. Includes support for:

  • Blinn-Phong shading
  • Normal mapping (including tangent generation)
  • Draw call sorting
  • Shadow mapping
  • Deferred shading

Edge of No World

Short platform game demo written in C++.

  • Uses asset pipeline to pre-process assets and pack them into a single file.
  • Uses Lua for gameplay scripting.
  • Uses a custom graph format for A* pathfinding.
